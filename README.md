# 🌏 Travel Journal

> **Travel Journal is a simple, lovely, and offline-friendly mobile app that helps travelers record expenses, memories, and emotions during every journey.**

---

# ✨ Vision

Create the easiest travel journal in the world.

Travel Journal is not only an expense tracker.

It is a memory book for every trip.

---

# ❤️ Mission

Help travelers remember every journey,

not only every expense.

---

# 📱 MVP Features

- ✈️ Trip Management
- 💰 Expense Tracking
- 📷 Photo Memories
- 😊 Emoji Journal
- 📝 Daily Notes

---

# 🎯 Product Philosophy

Travel Journal focuses on memories first.

Expenses are important.

But stories are priceless.

---

# 🎨 Design Principles

- Friendly
- Cute
- Pastel Theme
- Minimal
- Easy to Use
- Offline First
