# 🎨 UI Bible

Last Updated: 2026-07-XX

---

## 📖 Purpose

Define the visual language and user experience standards for Travel Journal.

Every screen should feel consistent, calm, and memory-focused.

This document acts as the single source of truth for UI and UX decisions.

---


## 🎯 Design Principles

### Memory First

Every screen should help preserve memories.

---

### Simplicity

Users should complete any action within a few taps.

---

### Emotional Design

Photos and memories should always receive more visual attention than numbers.

---

### Calm Interface

Avoid visual clutter.

Provide generous spacing.

---

### Offline First

Users should never worry about internet connectivity.
